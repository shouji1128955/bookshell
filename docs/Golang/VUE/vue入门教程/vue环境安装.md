



安装指定版本npm

```
npm install npm@6.0.0
```

npm config edit 

查看配置信息的指令：

```
npm config list 
```

设置淘宝镜像

```html
npm config set registry https://registry.npm.taobao.org
```

可以通过下面的命令查看淘宝镜像是否安装成功

```
npm config get registry
```

全局安装vue-cli

```html
npm install -g @vue/cli
```

查看vue-cli的版本

```
vue -V
```
