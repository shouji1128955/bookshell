## 1.代码拉取


Command line instructions
```shell
Git global setup
git config --global user.name "Administrator"
git config --global user.email "409003604@qq.com"

```

Create a new repository
```shell

git clone http://gitlab.kube-ops.svc.cluster.local/root/golang-test.git
cd golang-test
touch README.md
git add README.md
git commit -m "add README"
git push -u origin master
```


Existing folder
cd existing_folder
```shell
git init
git remote add origin http://gitlab.kube-ops.svc.cluster.local/root/golang-test.git
git add .
git commit -m "Initial commit"
git push -u origin master

```


Existing Git repository

```shell
cd existing_repo
git remote rename origin old-origin
git remote add origin http://gitlab.kube-ops.svc.cluster.local/root/golang-test.git
git push -u origin --all
git push -u origin --tags   
```
