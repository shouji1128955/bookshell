1、当前现状​

1.1、现状情况：​

由于之前clickhouse03节点磁盘直接打满，节点停止大概二十天时间，局限于扩容磁盘，因为其他节点清理历史的数据，所以腾出来了空间，我们现在需要对clickhouse03节点做完快照后，需要做数据同步.
